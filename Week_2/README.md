# 🚢 Titanic Dataset - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Titanic dataset using Python. The dataset was cleaned, analyzed, and visualized to identify important patterns related to passenger survival.

---

## ✨ Features

- 📂 Load the Titanic dataset
- 🧹 Clean missing values
- 📊 Perform Exploratory Data Analysis (EDA)
- 📈 Create visualizations using Matplotlib
- 💾 Save the cleaned dataset
